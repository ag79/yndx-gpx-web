Яндекс карты ⇒ GPX

Поддерживаются:

  • пользовательские карты, созданные в Конструкторе карт Яндекса.
  • проложенные Яндексом маршруты навигации (берётся только выбранный вариант маршрута)

Не поддерживаются созданные линейкой отрезки на обычной карте.

  • Маршруты: каждый отрезок будет отдельным маршрутом ("дистанция" или "course" в Garmin). Подходит для навигации с устройствами Garmin и большинством других.
  • Треки: каждый отрезок будет отдельным треком. Некоторые устройства поддерживают навигацию по треку.
  • Один трек: отрезки в исходном порядке будут склеены как сегменты в один трек с потерей названий.

Для аналитики, в навигации большинством устройств не используются. Увеличивает размер файла.
